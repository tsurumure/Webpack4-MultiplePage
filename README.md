## webpack4 快捷初始配置
Webpack v4.42.1
多入口 多页面 静态页 按需引用

#### 安装
```
$ yarn 
```

#### 使用
```
$ npm run dev
```

##### 生成文件目录
```
├─ dist
│  ├─ css
│  │  ├─ common.css
│  │  ├─ about.css
│  │  ├─ index.css
│  ├─ js
│  │  ├─ common.js
│  │  ├─ about.js
│  │  ├─ index.js
│  ├─ about.html
│  ├─ index.html
```
